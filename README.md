# Zodiak-Weton-Kotlin
Aplikasi Zodiak Weton Berbasis Mobile yang menggunakan Android Kotlin berguna untuk mengetahui info Zodiak dan Weton.

# Usage
* Install Java SE Development Kit
* Install Android Studio
* Configure Android Studio
* Open Android Studio
